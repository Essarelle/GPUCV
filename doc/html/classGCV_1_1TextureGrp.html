Detailed Description

A texture group contains a group of textures that can be used as input or output textures. Texture groups are used to manipulate several textures of a same type (input or output), it is used to defines multi-texturing input or output for filters. Some control flags can be used to check the all the texture from the group shares the same characteristics.

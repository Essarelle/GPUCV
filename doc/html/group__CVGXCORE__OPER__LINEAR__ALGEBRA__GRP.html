GPUCV correspondence of cvGEMM function.

Parameters:
src1 -> The first source array.
src2 -> The second source array.
src3 -> The third source array (shift). Can be NULL, if there is no shift.
dst -> The destination array.
alpha -> multiplication coef for (src1*src2).
beta -> multiplication coef for src3.
tABC -> The operation flags that can be 0 or combination of the following values: CV_GEMM_A_T - transpose src1 CV_GEMM_B_T - transpose src2 CV_GEMM_C_T - transpose src3

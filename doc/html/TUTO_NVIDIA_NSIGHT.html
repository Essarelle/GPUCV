In this tutorial, we will describe how to debug GpuCV applications using NVIDIA Parallel Nsight
  1. Setup
  2. Compile and run
  3. Debug

Setup

Download and setup NVIDIA Parallel Nsight host and monitor

Setup steps (Windows only):

  1. On Windows Vista/7: disable AERO desktop theme.
  2. Export <g>NSIGHT_CUDA_TOOLKIT</g> to "c:\Program Files\NVIDIA GPU Computing Toolkit\CUDA\v3.x"
  3. Open NSIGHT CUDA custom build rules ("c:\Program Files\Microsoft Visual Studio 9.0\VC\VCProjectDefaults\NsightCudaRuntimeApi.v3x.rules") and add the following include paths: 
    #Add following values after the [INCLUDE] field at line 10
-I&quot;$(CUDA_INC_PATH)&quot; -I&quot;$(NVSDKCOMPUTE_ROOT)/C/common/inc&quot; -I&quot;./&quot; -I&quot;../../../dependencies/otherlibs/include/&quot; -I&quot;../../../src/lib/GPUCV&quot; -I&quot;../../../include/&quot; -I&quot;../../../src/plugin/&quot;

  4. Refresh GpuCV solution for VS2008 to configure with fresh Nsight setup: 

    #in ./gpucv/ run:
premake4.exe vs2008

    TUTO_NVIDIA_NSIGHT__STP2__COMPILE_RUN Compile and run

  5. Export PATH for GpuCV: 
    #get path by running command:
premake.exe getpath
#export corresponding path
#ex: set PATH=%PATH%;%CD%\dependencies\otherlibs\bin\;%CUDA_BIN_PATH%;%OPENCV_PATH%\bin\;%NVSDKCOMPUTE_ROOT%\C\bin\win64\Release\;%NVSDKCOMPUTE_ROOT%\C\bin\win32\Release\;%OPENCV_PATH%\bin\Release;%CD%\dependencies\otherlibs\bin\x64;%CD%\dependencies\otherlibs\bin\x32;%CD%\lib\windows-vs2008-nsight\;%CD%\dependencies\otherlibs\bin\;%CD%\dependencies\SugoiTools\lib\windows-vs2008\;
  6. Start Nsight Monitor manually to benefit from the new PATH values: 
    "C:\Program Files\NVIDIA Parallel Nsight Monitor 1.0\Common\Nsight.Monitor.exe"
  7. Start Visual Studio: 
    devenv
  8. Open VS2008 project from "./gpucv/build/windows-2008/" and select configuration Debug_Nsight or Release_Nsight.
  9. Compile GpuCV, new binaries will be in "./gpucv/lib/windows_vs2008-nsight/" folder.
  10. Run your application as usual

Debug

Profile
In the new Analysis Activity windows, select the following values:

Field

Value

Application

g:-vs2008-Nsight\GPUCVConsole.exe

Arguments

loopnbr 5;resizeimg 1042 1024;disable debug;add;q;

Working directory

G:-vs2008-Nsight

Trace settings

Enable CUDA and OpenGL

Note:
It is recommanded to have 2 GPUs and to perform debugging on the one that do not have any screen connected.

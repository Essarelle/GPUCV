CvArr *  dst,
int flip_mode   CV_DEFAULT0 
)

GPUCV correspondence of cvFlip function. The function cvgFlip flip a 2D array around vertical, horizontall or both axis.

Parameters:
src -> Source array.
dst -> Destination array. If dst = NULL the flipping is done inplace.
flip_mode -> Specifies how to flip the array. flip_mode = 0 means flipping around x-axis, flip_mode > 0 (e.g. 1) means flipping around y-axis and flip_mode < 0 (e.g. -1) means flipping around both axises. See also the discussion below for the formulas

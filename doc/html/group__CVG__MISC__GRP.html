_GPUCV_EXPORT_C void cvgResizeGLSLFct ( CvArr *  src,
CvArr *  dst,
char *  GLSLfunction,
CvArr *mask   CV_DEFAULTNULL 
)

Similar to cvResize function but use the supply GLSL function to determine pixels results.

Parameters:
src -> source image
dst -> destination image
GLSLfunction -> GLSL function having two 'vec4' parameters, examples:

  • min - minimum nearest-neigbor,
  • max - maximum nearest-neigbor,
mask => mask array.
Warning:
Only the default interpolation method(CV_INTER_LINEAR) have been ported to GPU.
Author:
Yannick Allusse

GPUCV correspondence of cvThreshold function.

Parameters:
src => Source array (single-channel, 8-bit of 32-bit floating point).
dst => Destination array; must be either the same type as src or 8-bit.
threshold => Threshold value (see the documention).
max_value => Maximum value to use with CV_THRESH_BINARY and CV_THRESH_BINARY_INV thresholding types.
threshold_type => Thresholding type.
Author:
Integrated in GPUCV by Clement Beausset, code based on cvgThreshold().

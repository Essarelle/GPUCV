Detailed Description

This application is used to show real time processing of video stream using different processing implementations (Native OpenCV, GpuCV with GLSL, GpuCV with CUDA). See parseCommandLine() for command line options and CBkeys() for live key commands.

Warning:
ALL cv* function used into this demo application are rederected directly to GpuCV operators unless _DEMO_FORCE_NATIVE_OPENCV is set to 1. using the files "GPUCV/cv_wrapper.h", "GPUCV/cxcore_wrapper.h", "GPUCV/highgui_wrapper.h".
ALL the cvg* operator are pure GpuCV operators that have no equivalent in OpenCV.
Author:
Yannick Allusse
Version:
GpuCV 1.0.0 rev 562

Key

Description

from '1' to '9'

Select the correponding algorithm to run on video stream

ESC / 'Q' / 'q'

Exit the demo

'C' / 'c'

'p'

Enable/disable processing of the input video stream

'g'

Enable rendering the video ouput with OpenGL. Default is OPENCV render to CV window and GPUCV render to OpenGL window.

SPACE

Cycle processing implementation from OPENCV->GLSL->CUDA->OPENCV->...

'd'

Enable/Disable debugging log output. It impacts on performances.

's'

Enable/Disable switching log output. It impacts on performances.

'+'/'-'

Increase/Decrease loop number used by some processing algorithms.

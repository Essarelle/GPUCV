Member Enumeration Documentation

enum GCV::GpuCVSettings::Options
Enumerator:
GPUCV_SETTINGS_GL_ERROR_CHECK 

OpenGL error test are performed really often, which can slow down application.

GPUCV_SETTINGS_GL_ERROR_RISE_EXCEPTION 

OpenGL error will rise exceptions.

GPUCV_SETTINGS_LOAD_ALL_SHADERS 

Automatically load all shaders program from shader directories on the first call of cvgInit().

GPUCV_SETTINGS_USE_OPENCV 

Switch all the operators to use OpenCV if 1, GpuCV if 0.

GPUCV_SETTINGS_GLOBAL_DEBUG 

Enable the output of debugging informations to the console/log file.

GPUCV_SETTINGS_GLOBAL_ERROR 

Enable the output of error informations to the console/log file.

GPUCV_SETTINGS_GLOBAL_WARNING 

Enable the output of warning informations to the console/log file.

GPUCV_SETTINGS_GLOBAL_NOTICE 

Enable the output of standard informations to the console/log file, this flag should always be one unless you know what your are doing.

GPUCV_SETTINGS_CHECK_SHADER_UPDATE 

Used to check that the shader file has not been changed before running it.

GPUCV_SETTINGS_CHECK_IMAGE_ATTRIBS 

Default is enabled to check the CvgArr attributes(size, pixel format...) before applying filters.

GPUCV_SETTINGS_FILTER_DEBUG 

Default is enabled to check the CvgArr attributes(size, pixel format...) before applying filters.

GPUCV_SETTINGS_FILTER_SIMULATE 
GPUCV_SETTINGS_DEBUG_DATA_TRANSFER 
GPUCV_SETTINGS_SYNCHRONIZE_ON_ERROR 

Use to synchronize data with CPU and call OpenCV operator when blocking error occurs in GpuCV operators.

GPUCV_SETTINGS_PROFILING 

Enable runtime profiling, _GPUCV_PROFILE must be set to 1.

GPUCV_SETTINGS_PROFILING_TRANSFER 

Enable runtime profiling of data transfer, _GPUCV_PROFILE must be set to 1.

GPUCV_SETTINGS_PROFILING_OPER 

Enable runtime profiling of GPUCV operators, _GPUCV_PROFILE must be set to 1.

GPUCV_SETTINGS_PROFILING_CLASS 

Enable runtime profiling of GPUCV internal class, _GPUCV_PROFILE must be set to 1.

GPUCV_SETTINGS_DEBUG_MEMORY 

Enable runtime profiling of GPUCV internal class, _GPUCV_PROFILE must be set to 1.

GPUCV_SETTINGS_SWITCH_LOG 

Enable switch mechanism to be logged into console, see GPUCV_SWITCH_LOG.

GPUCV_SETTINGS_PLUG_THROW_EXCEPTIONS 

Enable plugin to throw exceptions to be catched by the main application.

GPUCV_SETTINGS_PLUG_AUTO_RESOLVE_ERROR 

Enable plugin to try to auto resolve error in an operator by selecting another implementation.

GPUCV_SETTINGS_LAST_ENUM

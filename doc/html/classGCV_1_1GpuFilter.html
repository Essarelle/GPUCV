Detailed Description

Manage a context (drawing function and parameter) for a specific filter.

Author:
Yannick Allusse
Erwan Guehenneux
Jean-Philippe Farrugia

This class centralize all informations between CPU and GPU and centralized them to simply apply on filter on a image.

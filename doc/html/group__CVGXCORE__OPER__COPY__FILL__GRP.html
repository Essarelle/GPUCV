Detailed Description

Contains CUDA correspondences of OpenCV cxcore operators.

Author:
Yannick Allusse Function to call numerous arithmetic and logics operators like Add/Sub/Mul...using mask image, or scalar values for the second source image.
Yannick Allusse
Warning:
Image depth and channel number can heavily impact on operator performances due to CUDA memory access mechanisms. This operator will reshape images to fit with coalescent memory read and write(it requires width * nchannels multiple of 2 or 4).
See also:
gcuUnsetReshapeObj(), gcuUnsetReshapeObj().

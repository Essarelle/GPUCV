Introduction

In this part we will present all files that do not belong to the project source code but are required/useful to compile/package/tweak/understand GpuCV.

File tree and categories

Here is a file/folder list with their descriptions and categories they belong to:

Path/Files

Description

Categories

Notes

/GpuCV/build/

/GpuCV/build/*

projects files for different targets

Project files

Generated by premake actions [vs2008|gmake|...]

/GpuCV/data/

/GpuCV/data/*

data files required to run GpuCV sample applications or ouput files of theses applications

Script files

/GpuCV/dependencies/

/GpuCV/dependencies/*

External dependencies headers/libraries required to compile/run GpuCV

Dependencies

/GpuCV/doc/

/GpuCV/doc/*

Doxygen styled documentation of GpuCV

Documentation

/GpuCV/etc/

/GpuCV/etc/*

script files related to external tools

Script files

/GpuCV/etc/Premake/*

Premake script files used to describe dependencies or OS, contains also actions and some usefull functions.

Packaging/Building

/GpuCV/etc/sed/*

SED rules files used to parse and clean OpenCV files when generating the OpenCV wrapper and switch files.

Switching

/GpuCV/etc/exclude*.lst

Exclude file/path list used by archive applications (tar/Winrar/7Zip/...). They are used to filter files when packagin GpuCV into zip/tar.gz distributions.

Packaging

/GpuCV/etc/*.rules

MS Visual Studio custom rules files used to compile and link *.cu files.

Building

/GpuCV/etc/cuda_common_*.lst

CUDA Linux makefiles.

Packaging

They are not updated automatically by premake. Do not forget to update them manualy if there are some compiling changes related to CUDA.

/GpuCV/src/

/GpuCV/src/*/*.lua

LUA files describing GpuCV libraries/plugins/applications

Project files

Files to edit to add a new plugin or test application

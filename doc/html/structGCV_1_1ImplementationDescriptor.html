Detailed Description

An implementation describe both a techology and a library/algorithm to implement new operators cause some operators can have several implementation using CUDA but using different libraries. Ex: cvgCudaAdd() and cvgNPPAdd() both implement the Add operator using CUDA, but the second one is using NVIDIA NPP library. They need to be distinguish by the switch so each new plugin will register itself and retrieve the associated implementation descriptor.
